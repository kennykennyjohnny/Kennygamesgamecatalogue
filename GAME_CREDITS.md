# 🎮 KennyGames - Game Credits

All games in KennyGames are adapted from open-source projects. We gratefully acknowledge the original creators:

---

## 🍾 LéaNaval (Battleship)

**Based on**: [Battleships](https://github.com/kubowania/battleships) by Ania Kubów

**Original Project**: Educational battleship game with multiplayer support

**License**: Not specified (educational project)

**Modifications**:
- Changed theme from military ships to wine bottles and cellar
- Implemented adjacency rule (bottles cannot touch)
- Adapted for asynchronous multiplayer with Supabase
- Custom rosé/bordeaux color palette
- Mobile-optimized drag & drop controls

**Credits**: Thank you to Ania Kubów for creating an excellent learning resource!

---

## 🎯 NourArchery (Cyber Archery)

**Based on**: [Archery Game](https://github.com/Adnan-Toky/archery-game) by Adnan Zawad Toky

**Original Project**: Canvas-based archery game with scoring system

**License**: MIT

**Modifications**:
- Added wind mechanics (random wind affecting trajectory)
- Changed theme to cyber/Matrix aesthetic with neon colors
- Implemented 3-round scoring system
- Adapted for turn-based multiplayer
- Enhanced visual effects (glowing targets, particle trails)
- Touch/swipe controls for mobile

**Credits**: Special thanks to Adnan Zawad Toky for the solid game foundation and MIT license!

---

## 🍷 SandyPong (Beer Pong)

**Based on**: [Beer Pong Canvas](https://github.com/FrBosquet/beerpong-canvas) by FrBosquet

**Original Project**: HTML5 Canvas beer pong game with physics

**License**: Not specified

**Modifications**:
- Changed theme from beer to rosé wine/terrace setting
- Replaced keyboard controls with swipe/drag controls
- Implemented game rules: rerack (6, 3, 2 cups), balls back (2 consecutive hits)
- Added bounce bonus mechanics
- Integrated asynchronous multiplayer
- Pink/rosé color palette for summer vibes

**Credits**: Thank you to FrBosquet for the excellent physics implementation!

---

## ⚡ LilianoThunder (Guitar Tanks)

**Based on**: [Scorch - Scorched Earth Clone](https://github.com/webermn15/Scorch_a-scorched-earth-clone) by webermn15

**Original Project**: Artillery game with destructible terrain

**License**: Not specified

**Modifications**:
- Replaced tanks with flying guitars/amplifiers
- Changed projectiles to lightning bolts
- Applied 80s neon aesthetic (magenta/cyan/yellow)
- Adapted terrain destruction for multiplayer sync
- Modified damage calculations
- Mobile-friendly controls (sliders for angle/power)

**Credits**: Thank you to webermn15 for the complex terrain destruction system!
